# Вариант C — «Чистый Swiss-минимализм» (фото-ориентированный)

> Перед стартом обязательно прочитай `briefs/_shared.md`. Этот файл — только про визуальную концепцию варианта C.

## Концепция

Объекты говорят сами за себя. Сайт — как швейцарский плакат: максимум воздуха, идеальная типографика, единственный акцентный цвет. Главное ощущение — **спокойствие, уверенность, качество без крика**.

Референсы по настроению (не копировать):
- сайты архитектурных бюро (Bürohaus, OMA, Herzog & de Meuron)
- The Browser Company, are.na
- швейцарская графика 60-х

## Палитра

| Роль | HEX |
|---|---|
| Фон | `#FFFFFF` (чистый белый) |
| Текст | `#0A0A0A` (почти чёрный) |
| Вторичный текст | `#7A7A7A` |
| Тонкие линии | `#E5E5E5` |
| Акцент | `#F24D20` (оранжевый из лого) — **единственный цвет** |

Только эти 5 значений. Никаких серых градаций больше. Никаких других оттенков.

## Типографика

- **Один шрифт:** `Inter` (или `Helvetica Neue` если доступен) — всё на сайте
- **Веса:** 300 (lighter), 500 (medium), 700 (bold) — этих трёх достаточно
- **Иерархия только через размер и вес, не через шрифт**

Размеры:
- H1: 80–96px, weight 500, line-height 1.0, letter-spacing -0.04em
- H2: 48px, weight 500
- H3: 24px, weight 500
- Body: 16px, weight 400, line-height 1.6
- Small: 13px, weight 400

## Сетка и воздух

- Container max-width 1280px, но контент часто шире/уже container'а — асимметрия приветствуется
- Padding между секциями: 160px на десктопе, 96px на мобиле
- Внутри секций — много пустого пространства

## Header

- Очень тонкий, высота 64px
- Лого слева
- Меню справа: пункты в `Inter 13px medium`, между ними разделитель `·`, без подчёркиваний
- На скролле — лёгкое размытие фона `rgba(255,255,255,0.85)` + `backdrop-filter: blur(20px)`
- Без теней, без рамок

## Hero

- На весь экран `100vh`
- Layout: 60/40 split
  - **Слева (60%)** на белом — заголовок крупно:
    > Строительство объектов федерального значения
  - Под заголовком короткий абзац (60 слов max)
  - Внизу — тонкая ссылка-стрелка `Все объекты →`, без кнопки
  - **Справа (40%)** — оранжевая плашка `#F24D20` во всю высоту с белым лого ССУ-138 по центру (или прижатым к низу)
- Никакого фотофона, никаких градиентов

## Услуги

- Очень простая секция
- 3 услуги вертикально, каждая занимает блок 200px высотой
- Между услугами тонкая линия `1px solid #E5E5E5`
- В строке: слева крупный номер `01` (вес 300, размер 64px), посередине название (вес 500, 32px), справа короткое описание + ссылка `Подробнее →`
- На hover строки — фон становится `#F24D20`, текст белым (плавно 300ms)

## Ключевые объекты

- Главная секция сайта по визуальному весу
- **Сторителлинг:** каждый объект занимает свой полный экран при скролле
- Layout одного объекта:
  - Сверху мелко (`Inter 13px`): порядковый номер `01 / 03`
  - Ниже — фотография 16:9 во всю ширину container (или даже шире, full-bleed)
  - Под фото — заголовок объекта крупно (48px)
  - Двухколоночный текст: слева описание, справа таблица фактов (заказчик, год, площадь, тип)
- Между объектами 200px воздуха

## Цифры

- Очень сдержанная секция
- 4 цифры в ряд, выровнены по baseline
- Каждая: цифра огромная (`Inter 300, 120px`), под ней подпись мелко
- Без иконок, без декора
- Между блоками — только воздух, без разделителей

## Благодарственное письмо

- На белом фоне
- По центру — скан письма, ширина max 600px, без рамки, без тени
- Под ним мелким `Inter 13px center`: «Благодарственное письмо · Источник, год»

## Этапы работы

- 5 шагов вертикально, как маркированный список — но без маркеров
- Слева моно-цифры `01 — 02 — 03 — 04 — 05`
- Линия `1px solid #E5E5E5` слева связывает шаги
- Каждый шаг: название (24px medium) + описание (16px regular)

## Форма + контакты

- Двухколоночная секция
- **Слева — форма:**
  - Поля без обводок и фонов, только подчёркивание снизу `1px solid #E5E5E5`
  - Лейблы исчезают при фокусе (плавающие)
  - Focus: подчёркивание становится `#F24D20`, толщина 2px
  - Кнопка `Отправить →` — текстовая ссылка с оранжевой стрелкой, без рамки
- **Справа — контакты:**
  - Адрес, телефон, email — обычным текстом
  - Карта — простой плейсхолдер 100% width, высота 320px

## Footer

- Очень тонкий, белый фон
- Сверху тонкая линия `1px solid #E5E5E5`
- Слева лого + © 2008–2026 ССУ-138
- Справа — повтор меню в одну строку
- Высота footer ~120px, никаких 4-колоночных простыней

## Анимации

- Минимум движения
- При скролле: `opacity 0 → 1` за 600ms (без translate)
- Hover на ссылках: подчёркивание появляется мгновенно
- На фотографиях: при наведении лёгкое изменение `filter: grayscale(0.2)` → `0` (неброско)

## Что обязательно

- Огромное количество пустого пространства
- Никогда не заполнять «пустоту» декоративно — она и есть дизайн
- Если убрать элемент и страница не сломается — убрать

## Что запрещено

- Тени любые
- Скругления больше 0px (всё прямоугольное)
- Иконки декоративные (только функциональные стрелки `→`)
- Градиенты
- Любой цвет, кроме белого, чёрного, серого `#7A7A7A`/`#E5E5E5` и оранжевого
- Анимации скейла, ротации, пружины
- Дополнительные шрифты помимо одного выбранного гротеска
